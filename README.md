# 🍿 Netflix India Landing Page Clone

> A visually rich, pixel-perfect front-end clone of Netflix India’s landing page using HTML and CSS. Built with responsive layouts, clean animations, and seamless mobile adaptation — designed to mirror the original feel of Netflix.

## ✨ Features

- 🎬 Hero section with background video overlay
- 📺 Sections for TV, mobile, and kids-friendly content
- 📦 Responsive dropdown FAQs (Accordion style)
- 🌐 Language selector and sign-in CTA
- 📱 Fully responsive and mobile-first design

---

## 🛠 Technologies Used

- HTML5  
- CSS3 (Flexbox, Grid, Media Queries, Animations)

---

## 💡 Why This Project?

Netflix’s branding and layout are both iconic and technically interesting. This clone was developed to sharpen responsive design skills and recreate a real-world UI with attention to detail.
